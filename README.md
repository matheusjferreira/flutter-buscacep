> ## The Project:

A simple application that allows you to consult zip codes..

![result](https://user-images.githubusercontent.com/59848966/189205286-d9c61b18-bc4f-4bfa-9e75-13a6f7f6817b.png)

* Darkmode;
* Error Handling;
* Local bank consumption;
